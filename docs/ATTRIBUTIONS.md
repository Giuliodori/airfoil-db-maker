# Sources, Attribution, and Redistribution Notes

This repository publishes code under `GPL-3.0-or-later`, but the generated database depends on upstream data sources and external tools that keep their own authorship and terms.

This document intentionally distinguishes between:
- what the upstream pages state explicitly
- what this repository does in practice
- what remains a redistribution check for the publisher

## Code License

- Repository code: `GPL-3.0-or-later`
- License file: [LICENSE](../LICENSE)

## Upstream Sources

### UIUC Airfoil Coordinates Database

- Source: UIUC Airfoil Coordinates Database
- URL: https://m-selig.ae.illinois.edu/ads/coord_database.html
- Role in this project: upstream airfoil coordinate source
- Related site overview: https://m-selig.ae.illinois.edu/ads.html
- The UIUC Airfoil Data Site states that the database includes airfoils from a wide range of applications and credits David Lednicer for a major expansion of the collection.
- The UIUC Airfoil Data Site asks contributors to identify the source of submitted airfoils.
- The UIUC Airfoil Data Site page carries the notice `© 1994 - 2026 UIUC Applied Aerodynamics Group`.
- No explicit standalone data redistribution license is identified in the project documentation at the URLs above.

### UIUC The Incomplete Guide to Airfoil Usage

- Source: UIUC The Incomplete Guide to Airfoil Usage
- URL: https://m-selig.ae.illinois.edu/ads/aircraft.html
- Role in this project: upstream aircraft/airfoil application references
- Related history page: https://m-selig.ae.illinois.edu/ads/history-ac.html
- The history page identifies David Lednicer, Aeromechanical Solutions LLC, as the maintainer of the guide.
- No explicit standalone redistribution license is identified in the project documentation at the URLs above.

### XFOIL

- Tool: XFOIL by Mark Drela and Harold Youngren
- Distribution page used by the build pipeline: https://web.mit.edu/drela/Public/web/xfoil/
- Role in this project: external solver used to generate polar data
- The XFOIL distribution page explicitly states that XFOIL is released under the GNU General Public License.
- The UIUC Airfoil Data Site also describes XFOIL as distributed under the GNU General Public License by Mark Drela and Harold Youngren.
- This repository does not commit `xfoil.exe`; the build pipeline downloads it separately if needed.

## Redistribution Notes

- Do not assume that upstream data archives, scraped content, or external binaries are relicensed under the repository GPL.
- XFOIL has an explicit GPL notice on its upstream distribution page, but that does not automatically apply to UIUC source data.
- For the UIUC sources referenced above, this repository currently relies on attribution and source preservation, not on an explicit relicensing grant identified in the source pages.
- The public `airfoil.db` artifact generated by this repository is sanitized before release and does not include raw upstream coordinate text or local filesystem paths.
- If you plan to redistribute source archives, mirrors, or materially expanded derivatives of the UIUC datasets, review the upstream pages again and contact the maintainers if you need a clearer redistribution position.
- Keep source attribution intact when publishing derived artifacts or repackaged datasets.

## Practical Publishing Position

- Repository code: publish under `GPL-3.0-or-later`.
- Generated public database: publish with clear upstream attribution and with no bundled XFOIL binary.
- XFOIL executable: do not commit it to this repository; let users obtain it from the official upstream source or through the build step.
- UIUC raw coordinate archives: do not mirror them in this repository unless you are comfortable with the upstream terms.
